About our Shopping List

This is a simple app, that allows you to create multiple lists and name them. You can also delete items from the list, or the whole list itself.

What is cool about it, is that it can work offline! You can create lists offline and whatever you had written down, will sync with the database when you are online again.

And even though it is a website, you can add it install it on your mobile device and it will behave like a native app.
